## 🌟 Overview
**Secure Academic Chain** is a modern Web3-inspired decentralized application designed to eliminate certificate forgery and tampering in educational institutions. By leveraging immutable cryptographic hashing and role-based access control, universities can securely issue verifiable credentials, students can access and manage their digital certificates, and third-party verifiers can instantly validate authenticity.

---

## ✨ Key Features

* **🏛️ University Admin Portal:** Secure authorization portal for university authorities to upload, record, and issue authenticated student credentials with automated blockchain hash generation.
* **🎓 Student Portal:** Dedicated dashboard featuring secure registration, custom password authentication, live credential status tracking, and direct certificate PDF downloads.
* **🔍 Public Verifier Portal:** Instant, trustless verification system allowing employers or institutions to check credential authenticity and ledger validity using a student's Register Number.
* **⚡ Modern Web3 UI/UX:** Sleek glassmorphism dark-mode interface built with custom CSS gradients, smooth transitions, and high-end aesthetic feedback.
* **🔗 Immutable Ledger Integration:** Every certificate is secured via unique cryptographic hashing to ensure absolute data integrity.

---

## 🛠️ Tech Stack

* **Frontend:** HTML5, CSS3 (Modern Glassmorphism Design System), JavaScript (ES6+)
* **Backend:** Node.js, Express.js
* **Database:** MongoDB Atlas (Mongoose ODM)
* **File Management:** Multer (for secure PDF certificate handling)
* **Security:** Cryptographic hashing protocols for immutable record checks
